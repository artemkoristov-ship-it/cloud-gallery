# Photo Gallery Web Application

## Опис проєкту

Photo Gallery Web Application — це односторінковий вебзастосунок для персоналізованого збереження фотографій користувачів.

Система дозволяє створювати власний обліковий запис, проходити авторизацію, завантажувати фотографії на сервер, переглядати власну галерею та видаляти непотрібні зображення.

Кожен користувач має доступ лише до власних фотографій, що забезпечує персоналізоване зберігання даних.

---

## Основні можливості

* Реєстрація користувачів.
* Авторизація користувачів.
* Зберігання JWT-токенів.
* Завантаження фотографій на сервер.
* Перегляд особистої галереї.
* Видалення фотографій.
* Захищений доступ до персональних даних.

## Встановлення проєкту

### Клонування репозиторію

git clone <repository-url>

cd gallery-app

---

## Встановлення залежностей

### Серверна частина

cd server

npm install

### Клієнтська частина

cd ../client

npm install

---

## Налаштування змінних середовища

У папці server створити файл .env

Приклад:

PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

---

## Запуск сервера

cd server

npm run dev

Після успішного запуску сервер буде доступний за адресою:

http://localhost:5000

---

## Запуск клієнта

cd client

npm run dev

Після запуску клієнтська частина буде доступна за адресою:

http://localhost:5173

## API маршрути

### Авторизація

POST /api/auth/register

Реєстрація нового користувача.

POST /api/auth/login

Авторизація користувача.

GET /api/auth/me

Отримання інформації про поточного користувача.

---

### Фотографії

POST /api/photos

Завантаження фотографії.

GET /api/photos

Отримання списку фотографій користувача.

DELETE /api/photos/:id

Видалення фотографії.
